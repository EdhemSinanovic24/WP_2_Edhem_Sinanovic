## 📌 O projektu

Projekat se sastoji od dva dijela:

1. **Tracker sistem** – moderna Angular web aplikacija za praćenje svakodnevnih navika
2. **IPI-Akademija** – kolekcija statičnih stranica i mini-aplikacija rađenih kroz semestar

---

## 🔄 Dio 1: Tracker sistem (Angular)

Pokretanje:

```bash
cd ipi-angular
ng serve
```

Aplikacija se pokreće na `http://localhost:4200`.

### Prijava i registracija
- Početna stranica nudi login i register formu
- Nakon registracije automatski se prelazi na login
- Nakon prijave otvara se **Dashboard**

### Dashboard sadrži 6 trackera

| Tracker | Opis |
|---|---|
| 💧 Water Tracker | Praćenje dnevnog unosa vode |
| 🍽️ Meal Tracker | Bilježenje obroka i hrane |
| 📅 Calendar Tracker | Kalendar za događaje i planove |
| 😊 Mood Tracker | Praćenje dnevnog raspoloženja |
| 🔄 Habit Tracker | Praćenje dnevnih navika |
| 😴 Sleep Tracker | Praćenje sna i spavanja |

> Podaci se čuvaju lokalno u browseru. Radi na svim modernim browserima.

## 🗂️ Dio 2: IPI-Akademija (statični projekat)

Bez instalacije – samo otvori `index.html` u browseru i sve radi.

### Struktura projekta

```
IPI-Akademija/
├── index.html          # Početna stranica
├── kontakt.html        # Kontakt forma (vizuelna)
├── popis.html          # Popis / informacije
├── raspored.html       # Raspored
├── studentfunzone.html # Pristup svim mini-igrama
├── bingo/              # Bingo igra
├── kviz/               # Kviz aplikacija
├── kanbanboard/        # Kanban board
├── visionboard/        # Vision board alat
├── whiteboard/         # Tabla za crtanje
├── css/                # Stilovi
├── slike/              # Slike i ikonice
└── mail/               # Kontakt forma (offline)
```

### Pokretanje

1. Otvori folder `IPI-Akademija`
2. Pokreni `index.html`
3. Sve ostale stranice i igre dostupne su iz menija

Kompatibilno s: Chrome, Brave, Edge, Firefox.
